When to Use Compare
Good use cases:
  • Before vs After firmware update — Did the update fix existing errors? Any new ones?
  • Working device vs broken device — Compare logs from a healthy camera to a failing one
  • Same camera, different days — Track if errors are getting worse over time
What you'll see:
  • New Errors — Errors that appeared in File 2 but not in File 1 (regressions)
  • Resolved Errors — Errors from File 1 that are gone in File 2 (fixes)
  • Line Diff — Unified diff of all error/warning lines
  • Severity Chart — Side-by-side bar chart of error counts
